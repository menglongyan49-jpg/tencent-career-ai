## 二、Agent 设计

### 2.1 Agent 基类

```python
class BaseAgent(ABC):
    def __init__(self, name: str, llm, personality: str, system_prompt: str):
        self.name = name
        self.llm = llm
        self.personality = personality
        self.system_prompt = system_prompt

    def build_system_prompt(self, base_prompt: str) -> str:
        """构建带性格的系统提示词"""

    @abstractmethod
    async def process(self, message: str, context: dict) -> AsyncGenerator[str, None]:
        """处理消息，流式返回响应"""
```

### 2.2 Memory Router

**职责**：
- 意图识别：分析用户输入的意图类型
- 情绪检测：检测用户情绪状态
- 任务分发：将请求路由给合适的专业 Agent
- 记忆更新：提取关键信息更新记忆

**路由规则**：
```python
ROUTING_RULES = {
    "emotion": "emotional_supporter",  # 情绪问题 → 情绪树洞
    "interview": "interview_coach",    # 面试问题 → 面试教练
    "tencent": "tencent_expert",       # 腾讯问题 → 鹅厂专家
    "career": "career_navigator",      # 职业问题 → 战术导师
    "technical": "career_navigator",   # 技术问题 → 战术导师
}
```

### 2.3 专业 Agent 职责

| Agent | 核心能力 | 输出特点 |
|-------|---------|---------|
| Career Navigator | 能力对标、路径规划、技术指导 | 硬核专业 |
| Emotional Supporter | 共情倾听、情绪疏导、正向引导 | 温暖共情 |
| Interview Coach | 简历优化、模拟面试、技巧指导 | 实战导向 |
| Tencent Expert | 业务解读、文化介绍、校招信息 | 权威准确 |

## 七、安全考虑

1. **API Key 保护**：环境变量存储，不提交到代码库
2. **输入验证**：对用户输入进行清理
3. **内容安全**：AI 输出内容审核
4. **数据保护**：用户数据本地存储，不上传云端

## 八、性能优化

1. **流式输出**：使用异步生成器实现流式响应
2. **记忆压缩**：只保留最近 N 条对话历史
3. **向量缓存**：对常见问题缓存检索结果
4. **懒加载**：知识库和向量库延迟初始化
