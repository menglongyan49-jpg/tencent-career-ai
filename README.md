# 「未来鹅」大学生职业成长 AI 陪伴体 (Multi-Agent 版)

## 项目概述

「未来鹅」是一款原生于 AI 的在校大学生求职成长陪伴智能体。本项目抛弃了传统的"被动信息检索"与"单向推送"模式，采用 **多智能体协同架构 (Multi-Agent System)** 与 **长期记忆机制 (Long-term Memory)**，打造具有千人千面性格的"拟人化"导师天团。我们将招聘前置为成长陪伴，在真实的情感链接中传递腾讯的雇主品牌价值。

---

## 一、核心破局点：AI Native 陪伴哲学

> **"真正的陪伴不是单向的千人一面，而是双向选择的契合。"**

### 1.1 性格"捏脸"系统
打破冰冷的客服感。学生可自定义陪伴体的沟通风格：
- 🎀 **知心学姐**：温柔细腻，善解人意，适合需要情感支持的同学
- 🤓 **硬核极客**：技术导向，直击痛点，适合追求技术深度的同学
- 👔 **严厉面试官**：模拟真实面试场景，一针见血，适合冲刺期同学
- 🌟 **全能导师**：平衡专业与温度，适合大多数同学

系统将动态重写底层 Prompt，实现定制化的人格陪伴。

### 1.2 多角色协同诊断
复杂问题不依赖单一模型幻觉。系统底层的多个 Agent 相互协同：
- **情绪安抚** → **能力诊断** → **路径规划** → **行动建议**
- 提供专业且有温度的反馈

### 1.3 长期记忆沉淀
陪伴体能记住：
- 学生的年级、专业、技术栈变化
- 项目经历的成长轨迹
- 情绪起伏的关键节点
- 让每一次对话都基于历史上下文"继续"，而非"重启"

### 2.1 多智能体天团 (Agent Roster)

| Agent | 职责 | 触发条件 | 输出特点 |
|-------|------|---------|---------|
| **🧠 Memory Router** | 记忆管理、意图路由 | 所有输入 | 结构化分发 |
| **🛡️ Career Navigator** | 求职路径、能力对标 | 技术问题、职业规划 | 硬核专业 |
| **❤️ Emotional Supporter** | 情绪疏导、心理支持 | 焦虑词汇检测 | 温暖共情 |
| **🎤 Interview Coach** | 模拟面试、简历优化 | 面试/简历关键词 | 实战导向 |
| **🐧 Tencent Expert** | 鹅厂文化、岗位解读 | 腾讯相关问题 | 权威准确 |

### 2.2 技术栈选型

| 层级 | 技术选型 | 选型理由 |
|-----|---------|---------|
| 前端交互 | **Streamlit** | 极简构建、原生流式支持、侧边栏配置 |
| Agent 引擎 | **LangChain + LangGraph** | 成熟的多Agent框架、状态管理 |
| 大模型 | **Claude API / GPT-4** | 强大推理、长上下文、流式输出 |
| 向量数据库 | **ChromaDB** | 轻量级、本地部署、快速检索 |
| 对话存储 | **SQLite** | 轻量级、无需额外服务 |
| 部署 | **Streamlit Cloud / Railway** | 一键部署、公网访问 |

## 三、核心交付场景 (Demo 体验流)

### 场景一：首次破冰 (Onboarding)
```
用户输入: 大三，计算机专业，想找后端开发实习，性格选"硬核极客"

系统响应:
→ 初始化用户档案 (grade: junior, major: CS, target: backend)
→ 动态生成极客风格欢迎语
→ 创建专属记忆空间
→ 开启对话
```

### 场景二：日常对练 (Daily Co-pilot)
```
用户输入: 我最近在学 Go 语言，但不知道怎么在简历上体现

Memory Router:
→ 意图识别: 技术学习 → 路由给 Career Navigator
→ 记忆更新: skills.append("Go")

Career Navigator (极客风格):
→ "Go 在腾讯云和微服务场景应用广泛..."
→ 指出大厂真实业务应用场景
→ 给出简历包装建议: "突出并发处理、微服务实践"
```

### 场景三：情绪疏导 (Emotional Care)
```
用户输入: 刚才面试挂了，好焦虑，感觉自己什么都不会

Memory Router:
→ 情绪检测: anxiety_level = high
→ 路由给 Emotional Supporter
→ 记录情绪事件

Emotional Supporter (知心学姐风格):
→ "面试挂了真的很打击，我理解你的感受..."
→ 共情倾听，缓解焦虑
→ 引导复盘，给出建设性建议
```
